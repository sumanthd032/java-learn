## Setup Instructions

1. Install the Java Development Kit (JDK) from Oracle or use OpenJDK.
2. Clone this repo:
   ```bash
   git clone <repo-url>
   ```
3. Navigate to the code folder:
   ```bash
   cd <folder-name>
   ```
4. Compile a Java file:
   ```bash
   javac FileName.java
   ```
5. Run it:
   ```bash
   java FileName
   ```
